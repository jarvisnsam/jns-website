Body Text

Lead paragraph - This is used for introductory text that needs more emphasis. It has a larger font size and line height for better readability.

Regular paragraph - Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Muted text - Used for secondary information, timestamps, or less important content.

Small text - Fine print, copyright notices, or auxiliary information.

Best Practices & Guidelines

Important guidelines for maintaining and extending the design system.

CSS Architecture

File Organization (style.css)
  1. CSS Variables - Theme configuration
  2. Base Styles - Body, typography defaults
  3. Typography - Heading and text styles
  4. Layout - Containers and sections
  5. Components - Reusable UI elements
  6. Forms - Input and form styling
  7. Buttons - Button variants
  8. Cards - Card components
  9. Navigation - Header and menus
  10. Footer - Footer styles
  11. Utilities - Helper classes
  12. Animations - Transitions and effects
  13. Responsive - Media queries
CSS Structure 
/* ==============================================
   SECTION NAME
   ============================================== */

/* Component description and usage notes */
.component-name {
    /* Structure properties first */
    display: flex;
    position: relative;
    
    /* Then sizing */
    width: 100%;
    padding: 20px;
    
    /* Then appearance */
    background: var(--bg-card);
    color: var(--text-primary);
    
    /* Finally transitions */
    transition: all 0.3s ease;
}

Development Guidelines

✅ Do's
  • Use CSS variables for colors and common values
  • Follow the existing naming conventions
  • Reuse existing utility classes
  • Test on multiple screen sizes
  • Comment complex CSS rules
  • Use semantic HTML elements
❌ Don'ts
  • Avoid inline styles
  • Don't use !important unless necessary
  • Don't create duplicate styles
  • Avoid overly specific selectors
  • Don't hardcode colors - use variables
  • Don't ignore mobile responsiveness
